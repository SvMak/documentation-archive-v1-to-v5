Welcome, Java Hipster!



JHipster is a Yeoman generator, used to create a Maven + Spring + AngularJS project, with full hot reload of Java and JavaScript code.

Our goal is to generate a complete and modern Web app, unifying:

  • Maven on the server side
  • Yeoman + Bower + Grunt on the client side

You can checkout a sample generated application here.

JHipster is Open Source, and all developement is done on Github

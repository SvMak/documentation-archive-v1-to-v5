JHipster release 0.15.0

JHipster gives you Yeoman + Maven + Spring + AngularJS all working together in one handy generator.

What’s new

JHipster 0.15.0 comes with two important updates:

  • A new “registration” page, that was one of our most requested features: #309
  • Corrected a database error that was occuring in production mode: #305

And of course several minor bugfixes and improvements, which are listed here

How to upgrade

Update your version of JHipster with:

npm update -g generator-jhipster

And then you can update your project when you run again

yo jhipster

Help and bugs

If you find any issue with this release, don’t hesitate to:

  • Contact @java_hipster on Twitter
  • Add a bug on our bug tracker
  • Post a question on Stack Overflow
